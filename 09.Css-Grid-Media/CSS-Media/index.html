How Do You Do?


Bu ilməquyruq meymun cinsi Amazonkanın cənub-şərqindəki yağışlı meşələrdə yaşayır. Xüsusi görüntüsü – sinəsinə qədər uzanan uzun ağ bığlar. Barmaqlarından yalnız arxa pəncəsinin baş barmağında dırnaq var. Həyatının böyük hissəsini ağacların ucqarlarında keçirirlər, hansı ki, sinfindən olan digər meymunlar həmin yerə çıxa bilmirlər. Bədənlərinin ölçüsü 9.2-10.4 düyüm, quyruğunun uzunluğu 14-16.6 düyüm olur. Ən böyüklərinin çəkisi 180-250 q.-dır.
